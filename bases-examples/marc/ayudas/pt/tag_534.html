Formato Bibliográfico MARC 21 LITE: Campos de Notas (Parte 1: 50X-53X)

Os campos 500-53X contém as notas relacionadas com os aspectos bibliográficos dos ítens que não são específicos para nenhum tipo de material ou controle (ex. controle para arquivamento). Os campos de notas especializados são utilizadas quando é necessário acesso aos dados  e/ou a nota é identificada por uma palavra ou frase. O campo 500 é utilizado para qualquer outro tipo de informação de notas.

  • 500 - NOTA GERAL (R)
  • 504 - NOTA DE BIBLIOGRAFIA (R)
  • 505 - NOTA DE CONTEÚDO FORMATADO (R)
  • 520 - RESUMO, ETC (R)
  • 521 - NOTA DE PÚBLICO ALVO (R)
  • 533 - NOTA DE REPRODUÇÃO (R)
[Mais informações em: Formato MARC 21 Conciso]

534 Nota de Versão Original (R)

Descrição de dados para o ítem original quando a parte principal do registro bibliográfico descreve a reprodução do item e os dados são diferentes. Detalhes relevantes para o original são fornecidos no campo 534

Indicadores

  • Primeiro – não definido
    • # - não definido
  • Segundo – não definido
    • # - não definido

Subcampos

  • ^a - Entrada principal do original (NR)
  • ^b - Informação de edição do original (NR)
  • ^c - Publicação, distribuição, etc. do original (NR)
  • ^e - Descrição física, etc. do original (NR)
  • ^p - Frase introdutória (NR)
    Uma frase que introduz a citação da versão original.
  • ^t - Informação de título do original (NR)

Exemplos

^534 ##^pOriginally published:^cNew York : Harper & Row, 1972,^b1st ed.
^534 ##^pOriginally published:^cNew York : Garland, 1987.
^534 ##^pOriginal version:^c[S.l. : s.n., 1832?] (New York : Mesier's Lith.).
^534 ##^pOriginally published:^cBerlin :Eulenspiegel, c1978,^tLieder zu Stücken.
