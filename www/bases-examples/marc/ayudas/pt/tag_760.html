Formato Bibliográfico MARC 21 LITE: Campos de Entrada de Ligações (76X-78X)

Os campos 760-787 contém informações que identificam outros itens bibliográficos. Cada entrada destes campos especifica uma relação diferente entre o item objeto descrito pelo registro e o item relacionado com ele. Estas relações são de três tipos:
  1. itens relacionados que ajudam ao usuário em uma busca mas que não são fisicamente requeridos para obter o item desejado (p.ex., entradas previas para seriados, tradução do item relacionado); itens relacionados que tem que ser obtidos fisicamente para poder utilizar o registro relacionado (p.ex. item hospedeiro que contém parte componente: um fascículo de uma revista que contém um artigo específico);
  2. itens relacionados que são unidades constituintes de uma totalidade maior (p.ex. fotografias individuais contidas em uma coleção de material visual).

Os campos de ligações de entradas têm o objetivo de gerar notas nas saídas nas quais aparecem, prover vinculações eletrônicas entre o registro bibliográfico do item objeto e o registro bibliográfico do item relacionado, no caso de o registro relacionado estar em um registro separado; e/ou facilitar a indexação.

  • 765 – ENTRADA DE IDIOMA ORIGINAL (R)
  • 767 – ENTRADA DE TRADUÇÃO (R)
  • 770 – SUPLEMENTO/ENTRADA DE EDIÇÃO ESPECIAL (R)
  • 772 – ENTRADA PRINCIPAL DE SUPLEMENTO (R)
  • 773 – ENTRADA DE ITEM (HOST) (R)

Registros de vínculos (links) e notas—informação general

Definição de termos usados nos campos de entradas de ligações (linking)

Item objeto (target)
um item bibliográfico que é a unidade principal ou primária para a descrição do qual o registro foi construído. O item objeto é o item ao qual o Leader/06 (Tipo de registro) e 07/ (Nível de registro) se aplicam. O título do item objeto está no campo do registro 245.


Item relacionado
um item bibliográfico que tem uma relação, seja cronológica, horizontal ou vertical, com o item objeto e para o qual um campo de ligação (linking) foi  formulado.


Parte componente
um item bibliográfico que fisicamente é parte de outro item bibliográfico de tal maneira que ao recuperar a parte esta depende da identificação física e localização do item hospedeiro (p.ex. um capítulo de um livro, um artigo de uma revista). (campo 774)


Unidade constituinte
um item bibliográfico que é parte de outro item. A unidade constituinte está fisicamente separada do item do qual é considerado parte. (Campos 770, 774)


Item hospedeiro (host)
um item bibliográfico que, ou  contém fisicamente a parte componente descrita pelo item objeto, ou inclui bibliograficamente a sub-parte que é o item objeto do registro. Por exemplo: se um registro descreve um artigo de revista, o hospedeiro é a revista, se um registro descreve um item digital em um objeto misto, o hospedeiro é a descrição do objeto misto. (Campos 772, 773)


Relações cronológicas
relações de tempo entre itens bibliográficos (p.ex., a relação de uma publicação seriada com seu predecessores e posteriores). (Campos 777, 780, 785)


Relaciones horizontais
a relação entre as versões de um item bibliográfico em diferentes idiomas, formatos, mídias, etc. (Campos 765, 767, 775, 776)


Relação vertical
a relação hierárquica do todo com suas partes e das partes como um todo (p.ex. um artigo de uma revista com a revista, uma subsérie com a entrada de série principal). (Campos 760, 762, 770, 772, 773, 774)

Campos de entrada de ligações (campos 760-787)
Estes campos contém dados descritivos do item relacionado, o número de controle para o registro do item relacionado ou ambos. Um conteúdo de designação mínima se da para os dados concernentes ao item relacionado que se usa nas entradas dos campos de ligação. Para operações de dados que requerem designação mais completa do conteúdo, tais como indexação ou ordenação, ou subcampo ^w (o número de controle do registro para o item relacionado) permite ao sistema seguir o vínculo até o registro relacionado para obter os dados completos.


Entradas secundarias (campos 700-730)
Quando um código de catalogação  requer uma entrada secundária de um título usado no campo de ligação, a entrada secundária se registra nos campos apropriados 700-730. Os campos de ligação não são destinados para entradas secundárias. Da mesma forma, uma entrada secundária no campo 700-730 não pode substituir um campo de ligação, pois não pode fazer que se gere una nota ou vínculo de um registro.


Partes Componentes
A entrada de ligação no campo 773 (entrada do item hospedeiro) se utiliza para vincular um item objeto que é parte (componente) integral de outro item com o registro do outro item. Por exemplo, em registros de artigos de periódicos, este campo contém a identificação do periódico. No subcampo ^g do campo 773, se informa a localização exata do artigo no periódico e gravado. O registro de um item hospedeiro ou alguma grande unidade bibliografica inclui informações sobre a unidade constituinte, armazenadas de um campo repetitivo 774 (Unidade constituinte de entrada). Informações sobre cada unidade constituinte será armazenada em ocorrências separadas no campo 774.

760 Entrada Principal de Série (R)

Informação relacionada com a série principal quando o item objeto é uma subsérie (relação vertical). Quando uma nota é gerada a partir deste campo, a frase introdutória Serie principal: ou Subsérie de: é gerada com base na informação da etiqueta (tag).

Indicadores
  • Primeiro – Controle de nota
    • 0 – Nota a ser exibida
  • Segundo – Controle de exibição de uma constante
    • # - Série principal

Subcampos
  • ^a - Entrada principal (NR)
    A entrada principal do campo 100 (Entrada principal - Nome pessoal), 110 (Entrada principal - Nome corporativo), 111 (Entrada principal - Nome de evento) do registro relacionado.
  • ^b - Edição (NR)
    Nome da edição do campo 250 (Edição) do registro relacionado.
  • ^d - Lugar, editora e data de publicação (NR)
    A informação de publicação dos subcampos ^a, ^b, e/ou ^c do campo 260 (Publicação, Distribuição, etc., ( Imprenta)) do registro relacionado.
  • ^g - Informação sobre relacionamento (R)
    Dados que indicam a peça ou peças especificas do item relacionado, envolvidos com o item objeto, tais como data e volumes. A informação de relacionamento inclui a localização de uma parte componente do item objeto.
  • ^h - Descrição física (NR)
  • ^i - Texto a exibir (NR)
  • ^n – Nota (R)
  • ^o - Outro item identificador (R)
  • ^t - Título (NR)
    O título dos subcampos ^a, ^n, e ^p do campo 245 (Informação de título) ou o subcampo ^a do campo 130 (Título uniforme) ou 222 (Título chave) do registro relacionado.
  • ^w - Número de Controle do Registro (R)
    O número de controle do sistema do registro relacionado precedido pelo código MARC, entre parêntesis, da agência à qual o número de controle se aplica. A fonte dos códigos é a Lista de Código MARC para Organizações ou uma lista afiliada.
  • ^x - ISSN (NR)
    O ISSN do registro relacionado, tomado do campo 022 (ISSN). O rótulo ISSN  pode ser gerado para ser apresentado.

Exemplos

760 0#^aUnited States. Geological Survey.^tWater supply papers
