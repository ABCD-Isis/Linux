Formato Bibliográfico MARC 21 LITE: Campos de Número e Código (01X-04X)

Os campos 010-04X contem os números de controle e de ligação, os números padronizados, e os códigos que relacionam o ítem bibliográfico descrito no registro.

  • 010 – NÚMERO DE CONTROLE DA BIBLIOTECA DO CONGRESSO USA (NR)
  • 015 – NÚMERO DA BIBLIOGRAFIA NACIONAL (R)
  • 017 – COPYRIGHT OU NÚMERO DE DEPOSITO LEGAL (R)
  • 020 – ISBN (R)
  • 022 – ISSN (R)
  • 024 – OUTROS IDENTIFICADORES PADRONIZADOS (R)
  • 028 – NÚMERO DO EDITOR (R)
  • 035 – NÚMERO DE CONTROLE DO SISTEMA (R)
  • 037 – FONTE DA AQUISIÇÃO (R)
  • 040 – FONTE DE CATALOGAÇÃO (NR)
  • 043 – CÓDIGO DE ÁREA GEOGRÁFICA (NR)

[Maiores informações em: Formato MARC 21 Conciso]

041 Código de Idioma (R)

Contem os códigos de idiomas MARC associados com o ítem quando o código do campo 008/35-37 é insuficiente para informação completa. Isto inclui registros de ítens multilíngue, ítens que envolvem tradução e ítens onde o meio de comunicação é uma linguagem de sinais. A fonte destes códigos é Lista de Código MARC de Idiomas ou outra lista de código como o ISO 639-1 (Códigos para a representação de nomes de idiomas - Parte 1 : código alpha-2)

Indicadores

  • Primeiro – Indicador de tradução
    • 0 = Item não é tradução/não inclui tradução
    • 1 = Item é ou inclui uma tradução
  • Segundo – Fonte de Código
    • # - Código MARC de idiomas
    • 7 - Fonte especificada no subcampo ^2

Subcampos

  • ^a - Código de Idioma para texto/trilha sonora ou título separado (R)
    O primeiro código de idioma do subcampo ^a também está no campo 008/35-37 (Idioma), a menos que as posições 35-37 contenham brancos (###) ou o código "zxx" (não há conteúdo linguístico).
  • ^h - Código de idioma da versão original e/ou traduções intermediárias do texto (R). Códigos de traduções intermediárias que precedem o código do idioma original.
  • ^2 Fonte do código (NR).

Exemplos

041

 0#^aen^agfre
041 1#^aeng^hger^hrswe
