Formato Bibliográfico MARC 21 LITE: Campos de Notas (Parte 1: 50X-53X)

Os campos 500-53X contém as notas relacionadas com os aspectos bibliográficos dos ítens que não são específicos para nenhum tipo de material ou controle (ex. controle para arquivamento). Os campos de notas especializados são utilizadas quando é necessário acesso aos dados  e/ou a nota é identificada por uma palavra ou frase. O campo 500 é utilizado para qualquer outro tipo de informação de notas.

  • 500 - NOTA GERAL (R)
  • 504 - NOTA DE BIBLIOGRAFIA (R)
  • 520 - RESUMO, ETC (R)
  • 521 - NOTA DE PÚBLICO ALVO (R)
  • 533 - NOTA DE REPRODUÇÃO (R)
  • 534 - NOTA DA VERSÃO ORIGINAL (R)

505 Nota de Conteúdo Formatado (R)

Os títulos de obras separadas ou partes de um ítem ou a tabela de conteúdo.  Este campo também  pode conter informações de responsabilidade, números de volumes e outras designações sequênciais.

Indicadores

  • Primeiro – Constante para apresentação
    Controlar a geração de uma frase introdutória
    • 0 - Conteúdo
  • Segundo – Nível de designação do conteúdo
    • # - Básico

Subcampos

  • ^a - Nota de conteúdo formatada (NR)
    O formato da nota é determinado pelas regras de catalogação.

Exemplos

505 0#^apt. 1. Carbon -- pt. 2. Nitrogen -- pt. 3. Sulphur -- pt. 4. Metals.
505 0#^aHow these records were discovered -- A short sketch of the Talmuds -- Constantine's letter.
