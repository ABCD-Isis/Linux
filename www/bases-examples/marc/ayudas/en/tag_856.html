Formato Bibliográfico MARC 21 LITE: "Holdings", Localização, Gráficos alternados, etc. Campos (841-88X)

Os campos 841-86X contém descrições de elementos que são parte integral do Formato Bibliográfico de MARC 21, e elementos de dados que podem aparecer, no mesmo registro bibliográfico, ou como registros MARC de coleções separadas.

  • 850 – INSTITUIÇÃO DEPOSITÁRIA (R)
  • 852 – LOCALIZAÇÃO (R)
  • 866 – HOLDINGS TEXTUAL - UNIDADE BÁSICA BIBLIOGRÁFICA (R)
  • 867 – HOLDINGS TEXTUAL - MATERIAL SUPLEMENTAR (R)
  • 868 – HOLDINGS TEXTUAL - ÍNDICES (R)

856 Localização Eletrônica e Acesso (R)

Contem a informação requerida para localizar um recurso eletrônico. O campo pode ser usado no registro bibliográfico de um recurso quando tal recurso, ou um sub-conjunto deste, está disponível eletronicamente. Adicionalmente, pode ser usado para localizar e acessar uma versão eletrônica de um recurso não eletrônico descrito no registro bibliográfico, ou um recurso eletrônico relacionado.

Veja as Orientações para o uso do campo 856 para maior aprofundamento sobre a utilização do campo 856.

O campo 856 é repetitível quando os elementos dos dados da localização variam (a URL no subcampo ^u ou subcampo ^a, ^b, ^d, quando usado). Também se repete quando mais de um meio de acesso é utilizado, diferentes partes do item estão disponíveis eletronicamente, espelhos de sites são armazenados, diferentes formatos/resoluções com URLs diferentes são indicadas, e itens relacionados são armazenados.

Indicadores

  • Primeiro – Método de acesso
    • # - não informado
      Indica que não há informação sobre o método de acesso. Este valor se usa quando o subcampo ^u contem uma URN (Nome Uniforme de Recurso) e não há URL registrada. Quando o subcampo ^u contem uma URL e o subcampo é repetido com uma URN, o valor do indicador para o método apropriado da URL é informado      .
  • Segundo – Relação
    Contem um valor que identifica a relação entre o recurso eletrônico identificado no campo 856 e o item descrito no registro como um todo.
    • # - Não informado

Subcampos

  • ^u - URL do Recurso (R)
    A URL prove a sintaxe padronizada para localizar um objeto utilizando os protocolos da internet. O campo 856 está estruturado para criar uma URL a partir da concatenação de outros subcampos separados do prórpio campo 856. O subcampo ^u pode ser usado em vez destes subcampos separados ou além destes. O subcampo ^u pode ser repetido somente se ambos URN ou uma URL ou mais que uma URN     são armazenadas.
  • ^x - Nota não pública (R)
  • ^z - Nota pública (R)
